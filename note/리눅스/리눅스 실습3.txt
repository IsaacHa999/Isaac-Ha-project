프로그램 :
 

 

 

 

scanf() 함수는 입력을 받는 함수이다. 그리고 이 함수의 결과 값은 성공적으로 입력된 데이터의 개수이다. 이 결과 값을 저장하기 위해서 다음 프로그램의 6번째 줄과 같은 문장을 사용하면 scanf()함수의 결과 값이 cnt 변수에 저장된다. 다음 프로그램을 실행해 보아라. (0.4점씩 총 2점)
#include <stdio.h>

int main(void)

{

int a1, a2, a3, cnt;

printf("정수 세 개를 입력 하세요 : ");

cnt = scanf("%d%d%d", &a1, &a2, &a3);

printf("입력된 데이터 개수 : %d\n", cnt);

return 0;

}

(b) 10 20 ?를 입력했을 때의 출력 결과에 대해 설명하여라.

 

 

 

(c) 10 ? 30을 입력했을 때의 출력 결과에 대해 설명하여라.

 

 

 

(d) ? 20 30을 입력했을 때의 출력 결과에 대해 설명하여라.

printf() 함수는 출력되는 데이터의 형식을 지정할 수 있게 한다. 출력 형식은 변환명세에서 %와 변환 문자 사이에 온다. 정수를 위한 출력 형식을 위해 다음 프로그램을 보자. int num = 111; 는 정수 값을 저장하기 위한 변수를 만들고 111를 변수에 저장하는 문장이다. (0.4점씩 총 2점)
 

#include <stdio.h>

int main(void)

{

int num = 111;

printf("12345678901234567890\n");

printf("===%d===\n", num);

return 0;

}
